# Sales Data Analysis

## Objective
To analyze sales data and identify trends, category performance, and regional sales insights.

## Dataset
Sample Superstore Dataset (Kaggle)

## Tools Used
- Excel

## Analysis Performed
- Data cleaning and preparation
- Total sales calculation
- Category-wise and region-wise analysis
- Monthly sales trend analysis

## Key Insights
- Certain months show peak sales
- Technology category performs best
- Some regions show lower sales and need improvement
